### 4. generate default nixos configuration

1. create config

```shell
nixos-generate-config --root /mnt
```

2. edit config

```shell
nano /mnt/etc/nixos/configuration.nix
```

2.1. add nix flake

    ```nix
    nix.settings.experimental-features = "nix-command flakes";
    ```

2.2. git

    ```nix
    environment.systemPackages = with pkgs; [git];
    ```

2.3. networkManager

    ```nix
    networking.networkmanager.enable = true;
    ```

2.4. LUKS device

    ```nix
    boot.initrd.luks.devices = {
      crypted = {
        device = "/dev/disk/by-uuid/<the uuid of the /dev/nvme0n1>";
        preLVM = true;
        allowDiscards = true;
      };
    };
    ```
