### 2. partitioning and formatting (UEFI)

1. identify disk

```shell
lsblk
```

2. wipe disk

```shell
sudo wipefs -a /dev/nvme0n1
```

#### partitioning

1. create GPT partition table

```shell
parted /dev/nvme0n1 -- mklabel gpt
```

2. add the boot partition. NixOS by default uses the ESP (EFI system partition) as its /boot partition.

```shell
parted /dev/nvme0n1 -- mkpart ESP fat32 1MB 512MB
parted /dev/nvme0n1 -- set 1 esp on
```

3. create root/primary partition

```shell
parted /dev/nvme0n1 -- mkpart root ext4 512MB 100%
```

#### optional: LUKS encryption

4. encrypt root partition

```shell
cryptsetup luksFormat /dev/nvme0n1p2

cryptsetup luksOpen /dev/nvme0n1p2 crypted
```

5. create PV

```shell
pvcreate /dev/mapper/crypted
```

6. create VG

```shell
vgcreate vg /dev/mapper/crypted
```

7. create LV

```shell
sudo lvcreate -l '100%FREE' -n nixos vg
```

#### formatting

1. format boot partition

```shell
mkfs.fat -F 32 -n BOOT /dev/nvme0n1p1
```

2. format root partition

```shell
mkfs.ext4 -L nixos /dev/vg/nixos
```

#### mount

1. mount root partition

```shell
mount /dev/disk/by-label/nixos /mnt
```

2. mount boot partition

```shell
mkdir -p /mnt/boot
mount -o umask=0077 /dev/disk/by-label/boot /mnt/boot
```

#### check UUID

1. get UUID of /dev/nvme0n1p2

```shell
lsblk --fs
```
