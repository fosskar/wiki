### 3. Generate default nixos configuration

1. Create config

```shell
nixos-generate-config --root /mnt
```

2. Edit config

```shell
nano /mnt/etc/nixos/configuration.nix
```

  2.1. Add user

    ```nix
    oscar
    ```

  2.2. Add nix flake

    ```nix
    nix.settings.experimental-features = "nix-command flakes";
    ```

  2.3. git

    ```nix
    environment.systemPackages = with pkgs; [git];
    ```
